# React 19 + TypeScript 游戏平台手机端

8Win 风格纯前端手机端首页，基于 Vite + React 19 + TypeScript。

## 启动

```bash
npm install
npm run dev
```

浏览器打开后，用 DevTools 切换到手机视图（建议 390×844）预览。

## 技术栈

- React 19
- TypeScript
- Vite 6
- lucide-react（图标）

## 页面结构

- 顶部 APP 下载推广条
- 导航栏（登录/注册/语言）
- Hero 轮播 Banner
- 最新获奖者（Tab + 横向滚动）
- 游戏分类网格
- 真人娱乐区
- 浮动客服按钮
- 底部导航栏
